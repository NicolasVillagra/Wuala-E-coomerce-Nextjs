## Instalación

Sigue estos pasos para configurar el proyecto localmente:

1. Clona este repositorio:

    ```bash
    git clone https://github.com/tuusuario/wuala.git
    ```

2. Navega al directorio del proyecto:

    ```bash
    cd wuala
    ```

3. Instala las dependencias:

    ```bash
    npm install
    ```
